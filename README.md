# Ra-El-amanecer

Este proyecto es un juego de aventuras basado en la mitología egipcia, donde los jugadores asumen el papel de ayudantes del dios Ra; el cual ha perdido sus poderes y debe recuperar sus habilidades para restaurar el equilibrio en el mundo contra Apofis, la serpiente del caos, antes de que la oscuridad eterna se apodere de todo. El juego está diseñado para ser educativo, enseñando a los jugadores sobre JavaScript a través de preguntas y desafíos que deben superar para avanzar en la historia.

## Características
- Explora un vasto mundo inspirado en el antiguo Egipto.
- Resuelve acertijos y completa misiones para recuperar los poderes de Ra.
- Cambio de fondo con cada nivel.
- Aprendizaje sobre JS con cada pregunta respondida correctamente.
  
## Requisitos del sistema
- Navegador web moderno (Chrome, Firefox, Safari, Edge).
- Conexión a Internet para descargar los archivos del juego.
- JavaScript habilitado en el navegador.
- Dispositivo con capacidad para ejecutar aplicaciones web (PC, Mac, tabletas, etc.).

## Herramientas utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Editor de código: Visual Studio Code
- Control de versiones: Git y GitHub
